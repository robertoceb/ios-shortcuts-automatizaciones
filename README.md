# 🚀 iOS Shortcuts & Automatizaciones

Colección de atajos y automatizaciones para iPhone orientadas a mejorar la experiencia diaria.

---

## 📲 Cómo usar este repositorio

1. Accede al atajo que te interese
2. Sigue su guía específica
3. Instala el atajo
4. (Opcional) Configura la automatización recomendada

---

## ⚙️ Conceptos clave

### 📲 Atajos

* Se pueden instalar directamente
* Son reutilizables
* Se pueden ejecutar manualmente o desde otros atajos

---

### ⚙️ Automatizaciones

* Se ejecutan automáticamente (evento → acción)
* **No se pueden exportar**
* Debes configurarlas manualmente
